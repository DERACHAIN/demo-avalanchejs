# Demo AvalancheJS

Example codes for AvalancheJS usage, including NodeID converter, NodePOP verification.

## Prerequisites

- Node v20

## Install

```sh
$ npm install
```

## Build

```sh
$ npm run build
```

## Run

- Convert NodeID to `bytes`

```sh
$ npm run start convertNodeId <nodeid-str>
```

- Verify Node POP

```sh
$ npm run start verifyPop <public-key-str> <pop-str>
```
